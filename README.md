## Hand Gesture Detector

This project introduces to learn Machine Learning about detect hand gesture from `Mediapipe` technology from google.

## What's I get this?

- This is for detect Hand landmarks, get 21 vector.
- Landmark have 5 code:
  - [0] -> Wrist
  - [1-4] -> Tip finger
  - [9-12] -> Middle finger
  - [13-16] -> Ring finger
  - [17-20] -> Pinky finger.
